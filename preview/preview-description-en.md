## Description by Tag

Add rich, shared context and notes to a tag so everyone understands what it represents.

**Why use it**

- **Document intent** — capture the purpose and rules behind each tag.
- **Single source of truth** — give your team one place to find the details.
- **Stay aligned** — keep group information consistent and discoverable.
