## Options by Tag

Apply shared configuration options to every record carrying a tag.

**Why use it**

- **Manage centrally** — control group-wide settings from one place.
- **Stay consistent** — keep configuration aligned across a team.
- **Change instantly** — update options for the whole group in one step.
